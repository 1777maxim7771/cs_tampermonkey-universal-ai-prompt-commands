# Tampermonkey Universal AI Prompt Commands CS

Tampermonkey skript pro nahrazení krátkých příkazů připravenými AI prompty v AI chatech.

Skript nahrazuje pouze přesné příkazy připravenými prompty. Běžný text se nemění. ChatGPT, Gemini, Claude, Copilot.

## Příkazy

- `CS1` — Přelož celý poskytnutý text přesně do češtiny.
- `CS2` — Shrň poskytnutý text v češtině podle významu a kontextu.
- `CS3` — Vytvoř v češtině krátké tematické shrnutí dopisu přesně na jeden řádek.
- `CS4` — Přelož poskytnutý text do jednoduché a srozumitelné němčiny na úrovni A2-B1.
- `CS5` — Oprav poskytnutý český text.
- `CS6` — Napiš krátkou, zdvořilou a úřední odpověď na tento dopis v jednoduché němčině úrovně A2-B1.
- `CS7` — Vysvětli jednoduchými slovy v češtině, co text znamená.
- `CS8` — Získej z textu všechny důležité skutečnosti a uspořádej je v češtině: osoby, organizace, adresy, data, lhůty, částky, čísla dokumentů, požadavky, rozhodnutí, povinnosti, dokumenty a další kroky.
- `CS9` — Sestav v češtině jasný seznam kroků podle textu.
- `CS10` — Na základě textu napiš zdvořilý úřední dopis v jednoduché němčině úrovně A2-B1.

## Instalace

Nainstalujte Tampermonkey, otevřete odkaz na userscript, potvrďte instalaci a zadejte první příkaz v AI chatu.

[Userscript](https://raw.githubusercontent.com/1777maxim7771/cs_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js)

Skript se instaluje do rozšíření Tampermonkey, nikoli do GitHubu ani na web. GitHub pouze ukládá soubor.

Tampermonkey → Dashboard → Utilities → Import from URL:

`https://raw.githubusercontent.com/1777maxim7771/cs_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js`
